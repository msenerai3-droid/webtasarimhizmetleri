Teknik Dokümantasyon

1. Sistem Mimarisi

YouTube Canlı Yorum Takip Sistemi, tamamen istemci tarafında çalışan bir web uygulamasıdır. Veritabanı işlemleri Firebase üzerinden yönetilir, API çağrıları YouTube Data API v3 kullanılarak yapılır.

Stack: HTML5 + CSS3 + Vanilla JavaScript + Firebase + YouTube API

2. YouTube API Entegrasyonu

Sistem, Google Cloud Console'da oluşturulmuş bir API key kullanır.

// Video ID'den canlı sohbet ID'sini al
const videoId = "dQw4w9WgXcQ";
const response = await fetch(`
https://www.googleapis.com/youtube/v3/videos?id=${videoId}&part=liveStreamingDetails&key=${apiKey}
`);
const activeLiveChatId = response.items[0].liveStreamingDetails.activeLiveChatId;

4. Veri Depolama Yapısı

Koleksiyon Döküman Alanlar
users {uid} email, displayName, photoURL, createdAt, trialStartedAt, hasFullAccess
config settings apiKeys[], trialDurationMinutes, quotaTotalMinutes, refreshInterval
records {recordId} title, type (first/sequential/highest/random), winners[], date

5. localStorage Yapısı

Yerel veriler tarayıcıda şu format ile saklanır:

// Kayıtlar
localStorage.setItem('yt_records', JSON.stringify(records));

// Dil tercihi
localStorage.setItem('appLang', 'tr');

// Çekiliş sonuçları
// Her çekiliş: { id, title, type, winners[], date }

6. API Rate Limiting

YouTube API'ın günlük limitleri:

Operasyon Günlük Limit Açıklama
Live Chat Mesaj Oku 100,000 Her mesaj 200 kredi tüketir
Video Detayı 100,000 Her sorgu 1 kredi tüketir
Toplam Günlük 1,000,000 Tüm API operasyonları

7. Hata Yönetimi

Sistem, yaygın hataları otomatik olarak yönetir:

✓ Geçersiz Video ID: Kullanıcıya bildirilir
✓ Canlı Yayın Bulunamadı: "Yayın aktif değil" mesajı
✓ API Hatası: Hata kodu gösterilir ve anahtar değiştirilir
✓ Bağlantı Hatası: Otomatik yeniden bağlantı denenir

8. Performans Optimizasyonları

✓ Comment Deduplication: Aynı yorum iki kez işlenmez (Set kullanımı)
✓ Scroll Virtualization: Çok sayıda yorum için performans
✓ Lazy Loading: Resimleri ve kaynakları isteğe bağlı yükleme
✓ Local Caching: Veriler tarayıcıda saklanır, sunucudan değil

9. Güvenlik Önlemleri

✓ XSS Koruması: HTML escape işlemi (escHtml fonksiyonu)
✓ CSRF Koruması: Firebase kimlik doğrulaması
✓ API Key Güvenliği: Key rotasyonu ve sınırlama
✓ Veri Şifreleme: HTTPS üzerinde iletim

10. Geliştirme için İpuçları

Yerel Test: `python -m http.server` ile localhost'da çalıştırabilirsiniz.
Debugging: Chrome DevTools Console ve Network sekmesini kullanın.
API Testing: Postman ile YouTube API'ı test edin.
Performance: DevTools Lighthouse ile performans kontrol edin.
